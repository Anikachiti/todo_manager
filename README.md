# To-Do List Manager

A beginner-friendly Python command-line To-Do List Manager that lets users add, view, delete, and mark tasks as done. Tasks are saved in a JSON file so they remain after the program is closed.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as done
- Save tasks in JSON format
- Basic error handling
- Supports tags and due dates

## Tech Stack
- Python
- JSON file handling

## How to Run
1. Make sure Python is installed.
2. Save the main file as `todo.py`.
3. Open terminal in the project folder.
4. Run the program:

```bash
python todo.py
```

## File Structure
```bash
todo-list-manager/
├── todo.py
├── tasks.json
└── README.md
```

## Notes
- `tasks.json` is created automatically when you add tasks.
- If the file does not exist, the program starts with an empty task list.
